Padding vs Margin vs Border

The margin property is the space around elements on a webpage, pretty much like the margin lines found in note books. It's the outermost part of any webpage element that 'separates' it from other elements around it. The border property is the space surrounding a webpage element, just after the margin. The property can be styled and made visible using CSS. The padding property is the space around the element just after the border, between the actual content and the border. This property can be used to 'cushion' the webpage element content within the border. Thus, every webpage element is surrounded by (in order) a: padding > border > margin.
